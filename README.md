# 🧠 Daily Python & SystemVerilog Practice

![Status](https://img.shields.io/badge/progress-active-brightgreen)
![Python](https://img.shields.io/badge/Python-Projects-blue)

A **30‑day engineering practice challenge** focused on building small
but meaningful projects in **Python** and **SystemVerilog**.

Each day includes:

-   🐍 **Python project** -- automation tools, data analysis, ML/AI
    mini‑projects, or developer utilities\
-   🔧 **SystemVerilog project** -- digital logic modules that ramp from basic
    gates to small hardware systems

All projects are intentionally **small (≈1--2 hours each)** to encourage
**daily consistency** and experimentation.

------------------------------------------------------------------------

# 🎯 Goals

This repository exists to:

-   Maintain **daily programming practice**
-   Strengthen both **software engineering** and **digital hardware
    design**
-   Build a collection of **small practical tools**
-   Explore areas like:
    -   automation scripts
    -   APIs and web scraping
    -   data analysis
    -   machine learning / AI
    -   developer utilities
    -   digital logic design

------------------------------------------------------------------------

# 📈 Skill Progression

## Python

Projects gradually increase in complexity:

CLI tools → automation scripts → web scraping & APIs → data analysis →
machine learning / AI → mini developer tools

Topics practiced:

-   Python scripting
-   APIs (`requests`)
-   Web scraping (`BeautifulSoup`)
-   Data analysis (`pandas`, `matplotlib`)
-   Machine learning (`scikit-learn`)
-   Automation tools
-   CLI utilities

------------------------------------------------------------------------

## SystemVerilog

Hardware exercises build from the ground up:

Logic gates → combinational circuits → arithmetic units → registers &
sequential logic → counters → finite state machines → small datapath
systems

Topics practiced:

-   combinational logic
-   adders and comparators
-   multiplexers
-   sequential circuits
-   counters
-   FSM design
-   basic hardware systems
